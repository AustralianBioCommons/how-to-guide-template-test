These are the steps required to make use of the contents of the How-to Guide template repository:

### 1. Create a new repository in your account / organisation, using [this repository]() as a template

### 2. Update the contents as described in the guides provided here

- [Create a new repository](how-to-guide-template/docs/create_new)
- [Update your landing page content](how-to-guide-template/docs/update_index)
- [Add content pages to your guide](how-to-guide-template/docs/add_new_pages)
   - [A single page template example](how-to-guide-template/docs/guide_template)
- [Update configuration files](how-to-guide-template/docs/structure)

### 3. [Deploy to GitHub pages](how-to-guide-template/docs/structure)
### 4. [Test, review and improve your guide content](how-to-guide-template/docs/improve_content)
### 5. [When ready create a GitHub release](how-to-guide-template/docs/zenodo)
### 6. [Register repository with Zenodo to create DOI](how-to-guide-template/docs/zenodo)
### 7. Share the How-to Guide!
